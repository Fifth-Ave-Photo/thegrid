# Grid Index RSS Importer

**Version:** 1.0.61  
**Type:** WordPress Plugin  
**Slug:** `grid-index-rss-importer`  
**Author:** Fifth Avenue Photographic  

Pull headlines from external RSS feeds into WordPress. Designed to pair with The Grid Index theme — imported posts are tagged with canonical source meta so the theme's "Read at Source" attribution lights up automatically. Works as a standalone importer if the theme isn't active.

## Installation

1. Download the latest release ZIP.
2. In WordPress admin, go to **Plugins → Add New → Upload Plugin**.
3. Upload the ZIP and click **Activate**.
